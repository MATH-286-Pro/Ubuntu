### DM-8009-2EC 电机参数

| 参数           | 值                                      |
| -------------- | --------------------------------------- |
| **双编码器**   | 是                                      |
| **额定扭矩**   | 20Nm                                    |
| **最大扭矩**   | 40Nm                                    |
| **额定转速**   | 100rpm (24V) / 200rpm (48V)             |
| **最大转速**   | 160rpm (24V) / 320rpm (48V)             |
| **电机重量**   | 896g                                    |
| **减速比**     | 9:1                                     |

#### 使用说明

> **工具:** 达妙USB2CAN  
> **注意:** 反线序CAN线连接 电机 与 USB2CAN + UART连接  
> 
> - 默认电机 `ID = 1`
> - 发送 `FF FF FF FF FF FF FF FC` 电机使能 (红灯常亮)  
> - 发送 `FF FF FF FF FF FF FF FD` 电机失能 (绿灯常亮)
